# Sistema De Cadastro de Produtos

> Statua do projeto: Em desenvolvimento

para rodar esse projeto no seu computador, por favor digite no terminal:

```
npm install react
```

Agora eu vou escrever um codigo em HTML para testar:

```
<!doctype html>
<html>
   <head>
    <meta charset="utf-8"/>
    <title>Sistema</title>
  </head>
  <body>
    <p>Olá mundo!</p>
  </body>
</html>
```
